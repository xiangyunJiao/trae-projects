## 1. 产品概述

H5魔法卡牌翻牌抽奖游戏——用户在暗色桌面上翻卡牌、洗牌、选牌，收集6张不同魔法卡牌赢取神秘大奖。目标用户为移动端H5休闲游戏玩家，核心价值在于翻牌的惊喜感和集齐全套卡牌的成就感。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 翻牌抽奖、查看背包、领取大奖 |

### 2.2 功能模块
1. **卡牌游戏页**：6张魔法卡牌展示、翻卡牌按钮、洗牌动画、选卡交互、翻牌结果弹窗
2. **背包页**：查看已获得的卡牌及数量、集齐大奖提示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 卡牌游戏页 | 桌面背景 | 暗色桌布纹理背景，营造魔法氛围 |
| 卡牌游戏页 | 卡牌区域 | 6张魔法卡牌正面平铺展示，每张图案不同 |
| 卡牌游戏页 | 翻卡牌按钮 | 点击后卡牌翻转为背面，触发洗牌动画 |
| 卡牌游戏页 | 洗牌动画 | 4秒洗牌，由慢到快，卡牌互换位置，流畅位移 |
| 卡牌游戏页 | 引导手指 | 洗牌结束后显示小手指引导用户选牌 |
| 卡牌游戏页 | 翻牌弹窗 | 用户点击卡牌后翻转显示正面，半透明弹窗展示获得卡牌 |
| 背包页 | 卡牌列表 | 展示已获得的所有卡牌及数量统计 |
| 背包页 | 大奖提示 | 集齐6张不同卡牌后显示神秘大奖 |

## 3. 核心流程

用户进入游戏页面 → 看到6张正面朝上的魔法卡牌 → 点击"翻卡牌"按钮 → 卡牌翻转为背面 → 4秒洗牌动画（由慢到快）→ 洗牌结束，6张背面卡牌平铺 → 出现引导手指 → 用户点击选择一张卡牌 → 该卡牌翻转为正面 → 半透明弹窗展示获得卡牌 → 卡牌存入背包 → 用户可继续翻牌或查看背包 → 集齐6张不同卡牌 → 获得神秘大奖

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深紫色/暗蓝色魔法氛围色（#1a0a2e、#16213e）
- 辅助色：金色高光（#ffd700）、魔法蓝光（#4fc3f7）
- 按钮风格：圆角渐变按钮，带魔法光晕效果
- 字体：魔法风格的展示字体 + 清晰的UI字体
- 布局：移动端居中布局，卡牌网格2行3列排列
- 图标/emoji风格：魔法元素（✨、🃏、🏆）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 卡牌游戏页 | 桌面背景 | 深色渐变、微妙纹理、魔法粒子效果 |
| 卡牌游戏页 | 卡牌区域 | 2行3列网格、卡牌有3D翻转效果、每张卡牌独特魔法图案 |
| 卡牌游戏页 | 翻卡牌按钮 | 金色渐变按钮、魔法光晕、点击缩放反馈 |
| 卡牌游戏页 | 洗牌动画 | CSS transform位移动画、贝塞尔曲线缓动、由慢到快 |
| 卡牌游戏页 | 引导手指 | 小手指向下方卡牌、上下弹跳动画 |
| 卡牌游戏页 | 翻牌弹窗 | 半透明遮罩、卡牌居中放大、金色边框光效、获得文案 |
| 背包页 | 卡牌列表 | 网格布局、卡牌缩略图、数量角标、未获得卡牌灰色 |
| 背包页 | 大奖提示 | 金色特效、奖杯图标、庆祝动画 |

### 4.3 响应式设计
- 移动端优先设计（H5页面）
- 卡牌尺寸自适应屏幕宽度
- 触摸优化：卡牌点击区域足够大
- 竖屏布局为主

### 4.4 3D场景指引
- 不适用，本项目使用CSS 3D变换实现卡牌翻转效果
