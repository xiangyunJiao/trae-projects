# 儿童拼音学习小程序

这是一个帮助学龄前儿童学习拼音的 H5 网页应用，支持语音识别评分。

## 技术栈
- **前端框架**: 纯 HTML5 + CSS3 + JavaScript（轻量，无需构建）
- **语音识别**: Web Speech API（浏览器原生）
- **语音合成**: Web Speech API（浏览器原生 TTS）
- **数据存储**: localStorage（本地存储学习记录）
- **UI 框架**: 无外部依赖，纯 CSS

## 功能模块

### 1. 生韵母发音学习
- 声母列表（23个）：b, p, m, f, d, t, n, l, g, k, h, j, q, x, zh, ch, sh, r, z, c, s, y, w
- 韵母列表（24个）：
  - 单韵母（6个）：a, o, e, i, u, v
  - 复韵母（9个）：ai, ei, ui, ao, ou, iu, ie, ve, er
  - 前鼻韵母（5个）：an, en, in, un, vn
  - 后鼻韵母（4个）：ang, eng, ing, ong
- 点击发音，支持跟读练习

### 2. 组合发音学习
- 两拼音节（声母+韵母）
- 三拼音节（声母+介母+韵母）
- 整体认读音节（16个）：zhi, chi, shi, ri, zi, ci, si, yi, wu, yu, ye, yue, yuan, yin, yun, ying

### 3. 学龄前汉字学习
- **分级体系（5个等级）**:
  - Level 1（入门级）：简单独体字、常用数字、方位等，约100字
  - Level 2（基础级）：日常事物、人体部位等，约100字
  - Level 3（进阶级）：自然现象、动作等，约100字
  - Level 4（提高级）：形容词、状态等，约100字
  - Level 5（拓展级）：抽象概念、时间等，约100字

- **分组学习**: 每10个字为一组
- **打卡系统**: 每日打卡记录
- **已学会标记**: 手动标记已掌握的汉字

### 4. 考试模式
- **考试类型**:
  1. 生韵母考试
  2. 组合发音考试
  3. 汉字发音考试

- **难度选择**: 简单/中等/困难
- **考试流程**:
  1. 随机抽取10-20题
  2. 儿童听题/看题后发音
  3. 系统语音识别判断是否正确
  4. 每题正确得10分，错误0分
  5. 最终百分制成绩，60分及格

- **成绩展示**: 得分、正确率、错题回顾
